자바스크립트에서는 `Primitive Value` (원시값)는 `Immutable`한 성격을 가진다. Immutable이란 수정할 수 없다는 뜻이다. 즉 접근 및 읽기는 가능하나 값을 변경하는 것은 불가능하다.

즉, 문자열을 indexing으로 접근 및 읽기는 가능하지만 `str[i] = 'a'` 이런 식으로 재할당은 불가능하다.
